TO LAUNCH THE DEVELOPMENT SERVER:
================================================================================

OPTION 1: Using PowerShell (Recommended)
-----------

1. Open PowerShell or Command Prompt
2. Navigate to project directory:
   cd "c:\Users\ZaidS\HealthMap Backend\HealthMap-backend"

3. Set up environment (first time only):
   $php = "C:\Laragon\bin\php\php-8.3.30-Win32-vs16-x64\php.exe"
   & $php artisan key:generate
   & $php artisan migrate

4. Start the development server:
   & $php artisan serve

5. Open browser to: http://127.0.0.1:8000

OPTION 2: Using the Helper Script
-----------

1. Open Command Prompt or PowerShell
2. Navigate to the parent folder:
   cd "c:\Users\ZaidS\HealthMap Backend"

3. Run the helper script:
   run-laravel.bat serve

4. Open browser to: http://127.0.0.1:8000

OPTION 3: Using Laragon GUI
-----------

1. Open Laragon application
2. Click "Start All" to start services
3. Navigate project to the www folder or use terminal in Laragon
4. Run: php artisan serve

================================================================================
TROUBLESHOOTING:
================================================================================

Issue: "php command not found"
Solution: Use full path: C:\Laragon\bin\php\php-8.3.30-Win32-vs16-x64\php.exe

Issue: "Application key has not been set"
Solution: Run: php artisan key:generate

Issue: "Database connection error"
Solution: Check .env file and ensure MySQL is running via Laragon

Issue: "Port 8000 already in use"
Solution: Use different port: php artisan serve --port=8001

Issue: Missing PHP extensions
Solution: Check PHP config at: C:\Laragon\bin\php\php-8.3.30-Win32-vs16-x64\php.ini
