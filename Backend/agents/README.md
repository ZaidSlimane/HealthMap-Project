# Agents

Local-only working notes produced by AI coding agents while working on this
repository. **Not pushed to remote** (see `.gitignore`).

## Contents
- `session-log.md` — chronological summary of changes made in agent sessions
- `bugs-fixed.md` — catalog of real bugs uncovered and how they were fixed
- `architecture-notes.md` — observations about the codebase structure
- `integration-guide.md` — frontend ↔ backend integration reference
- `controllers-scaffold.md` — notes on the ClinicalCore controller scaffold

These files are working memory. They can be deleted at any time without losing
anything that ships in the project.
